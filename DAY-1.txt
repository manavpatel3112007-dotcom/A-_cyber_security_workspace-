TASK 3 :  short explanation-CPU , RAM , GPU


CPU (Central Processing Unit) :The CPU runs the game rules. It tracks where you are running, calculates when you shoot an enemy, and controls how bots move around the map.

RAM (RANDOM ACCESS MEMORY   :RAM is a super-fast temporary storage space. It holds the map and the gun sounds you are using right now so the computer can open them instantly without lagging.

GPU (Graphics Processing Unit):The GPU makes everything look pretty. It draws the actual 3D graphics, the grass, the characters, and the explosions on your screen so the game moves smoothly.


TASK 4 :

1. The Graphics Card (GPU)

It handles everything you see on the screen—the player faces, the stadium lights, the grass, and the smooth movement of the ball.

2. The Processor (CPU)

It acts as the "brain" of your PC. In FIFA, it calculates player AI (how opponents move), referee decisions, ball physics, and crowd reactions.

3. RAM (Memory)

It holds temporary data that the game needs to access instantly while you are playing.

4. Storage: Move to an SSD (Solid State Drive)

This is where the game files are permanently saved.
